# Cyprus Realty Brandbook

Brand color analysis for the Cyprus real estate market — competitive color mapping to find a unique brand color.

## Pages

- **[Cyprus Only](cyprus-only.html)** — 18 Cypriot platforms and agencies
- **[International](international.html)** — Cyprus + 20 global real estate brands

## Data

All brand data in [`data/brands.json`](data/brands.json).

## Purpose

Mapping competitor brand colors on a color wheel to identify free hue zones for a new Cyprus real estate brand that stands out both locally and internationally.
